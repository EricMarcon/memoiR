# memoiR: R Markdown and Bookdown Templates to Publish Documents

Producing high-quality documents suitable for publication directly from
R is made possible by the R Markdown ecosystem. 'memoiR' makes it easy.
It provides templates to knit memoirs, articles and slideshows with
helpers to publish the documents on GitHub Pages and activate continuous
integration.
